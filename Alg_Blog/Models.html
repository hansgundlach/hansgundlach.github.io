Models of Computation

Intro to Random Acess and Pointer Machine

What are models of computation ?

Random Acess Machine

Random Acess Machines are built on a few basic principles. In princple a random acess machine is a big array.

Pointer Machine

A pointer machine dynamiclly allocates objects. A pointer machine is basically a big directed graphs with pointers as arrows.

Comparison Model of Computation or Decision Tree

AS the names suggest the comparison model of computation is based on the fact that any program can be written as a series of decisions. This model is especially useful for proving low bounds on computational complexity, For example we can establish a lower bound for any sorting algorithmn . We know a search can be considered as a binary decision tree with n! leafes. The height of this tree is approximatly log(n!) .
